Flood kahoot game remotely

## Url query
###### [*] example url: http://localhost:8080/?game=980268&basename=amit&count=20
###### [*] game=(int) Game Pin 
###### [*] count=(int) how many clients to addd
###### [*] basename=(string) your base name (amit) = amit + x
